# integer-puzzle
Node.js integer puzzle

Commandline application to take numeric input (N) and output a table of prime number.

Dependencies
------------
[Node] [NodeDownload]

Please note this application was developed on Windows 10

Install and Run
---------------

###Install

run node.js command prompt terminal

clone this repo

>npm install
